🔵 Segunda-feira: Superior Empurrar (PEITO, OMBRO E TRICEPS)

Frequência:Toda segunda (Duração: 2h)

1. Supino Inclinado com Halteres: 3 séries até a falha (90s descanso)
2. Supino Inclinado Artuculado Maquina: 3 séries até a falha (60-90s descanso)
3. Crucifixo Maquina:3 séries até a falha (60s descanso)
4. Elevação Lateral: 3 séries até a falha (60s descanso)
5. Desenvolvimento Maquina: 3 séries até a falha (45s descanso)
6. Tríceps Polia: 3 séries até a falha (45s descanso)
6. Tríceps Frances Polia: 3 séries até a falha (45s descanso)

🟢 Terça-feira: Inferior, Estabilizador A (QUADRICEPS, PANTURRILHA E PRANCHA)

Frequência:Toda terça (Duração: 2h)

1. Agachaento Livre ou Maquina: 3 séries até a falha (90s descanso)
2. Agachamento Pendulo 3 séries até a falha (90s descanso)
3. Leg 45°: 3 séries até a falha (60s descanso)
4. Hack Agachamento Frontal e Posterior: 3 séries até a falha (60s descanso)
5. Super Drop Passada Ida + Panturrilha Sentado + Panturrilha em Pé + Volta Passada: 3 séries até a falha (60s descanso)
6. Prancha: 3 séries até a falha (60s descanso)

🔵 Quarta-feira: Superior Puxar (COSTAS, PST OMBRO E BÍCEPS)

Frequência:Toda Quarta (Duração: 2h)

1. Puxada Neutra Articulada Maquina: 3 séries até a falha (90s descanso)
2. Puxada supinada Articulada Maquina: 3 séries até a falha (90s descanso)
3. Incline Lever Row (Remada Cavalinho): 3 séries até a falha (60s descanso)
4. FACEPULL (Puxada Na Cara polica com corda): 3 séries até a falha (60s descanso)
5. crucifixo Invertido Maquina: 3 séries até a falha (60s descanso)
6. Rosca Alternada com Halteres: 3 séries até a falha (60s descanso)
6. Rosca Declinada no Bancop 3 séries até a falha (60s descanso)

🟢 Quinta-feira: Inferior, Estabilizador B (POSTERIOR DE COXA, GLÚTEOS E ABS)

Frequência:Toda Quinta (Duração: 2h)

1. Mesa Flexora:Drop 6,8,10,12 e 15 (90s descanso)
2. Cadeira Flexora 3 séries até a falha (90s descanso)
3. Cadeira Abdutora: 3 séries até a falha (60s descanso)
4. Flexora Unilateral Health Maquina: 3 séries até a falha (60s descanso)
5. Stiff Belt Squat: 3 séries até a falha (60s descanso)
6. Sumô Belt Squat: 3 séries até a falha (60s descanso)
7. Abdominal Supra: 3 séries até a falha (60s descanso)

🔵 Sexta-feira: Hibrido Suplementar (PEITO, COSTAS E BRAÇO)

Frequência:Toda Sexta (Duração: 2h)

1. Supino Reto Máquina: 3 séries até a falha (90s descanso)
2. Supino Declinado Máquina: 3 séries até a falha (90s descanso)
3. Remada Curvada na Máquina: 3 séries até a falha (60s descanso)
4. Puxada Polia com Triangulo: 3 séries até a falha (60s descanso)
5. Rosca Pronada Barra W: 3 séries até a falha (60s descanso)
6. Tríceps Unilateral Polia: 3 séries até a falha (60s descanso)
